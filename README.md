# Insurance Dataset EDA

This project performs Exploratory Data Analysis (EDA) on an insurance dataset to understand the distribution of key features and prepare the data for machine learning models.

## Dataset Features
Age
Sex
BMI
Children
Smoker
Region
Charges

## EDA Visualization

![EDA Dashboard](Charts.png)

## Key Insights

Age distribution shows a balanced spread across different age groups.
BMI distribution follows a near-normal pattern.
Most customers have fewer children, making the distribution right-skewed.
